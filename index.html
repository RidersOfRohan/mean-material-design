Welcome to MEAN-MATERIAL-DESIGN

MEAN Material Design is a full-stack JavaScript open-source solution which is based on the MEAN.JS framework, which provides a solid starting point for MongoDB, Node.js, Express, and AngularJS based applications. The idea is to solve the common issues with connecting those frameworks, build a robust framework to support daily development needs, and help developers use better practices while working with popular JavaScript components. Whilst incorporating the latest Material Design concepts.
